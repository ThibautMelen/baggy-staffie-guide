## 📖 À Propos

Site web interactif créé pour Thibaut & Lucile, un couple parisien du 2ème arrondissement qui se prépare à adopter Baggy, leur futur Staffordshire Bull Terrier. Ce projet combine informations pratiques, outils interactifs et éléments ludiques pour une préparation optimale à l'adoption.

## ✨ Fonctionnalités

### 🎥 Contenu Multimédia
- **Vidéo Hero** : Présentation vidéo de Lucile avec Baggy
- **Galerie Photos** : Collection d'images de Baggy (chiot et adulte)
- **Vidéos Éducatives** : 8+ tutoriels YouTube intégrés sur l'éducation canine

### 🎮 Éléments Interactifs
- **Chien Interactif** : Caressez Baggy virtuellement avec effets visuels et musique Spotify
- **Quiz de Connaissances** : Testez vos connaissances sur la race
- **Checklist de Préparation** : Liste interactive pour valider votre préparation
- **Mini-Jeux** :
  - 🧠 Memory Game
  - 💰 Calculateur de Budget
  - 🏷️ Générateur de Noms
  - ⏱️ Quiz Chronométré
  - 🐾 Compteur de Caresses
  - 📅 Tracker de Préparation

### 📊 Informations Pratiques
- **Caractéristiques de la Race** : Taille, poids, espérance de vie, origine
- **Évaluation de Compatibilité** : Score personnalisé pour votre situation
- **Planning Quotidien** : Journée type avec un Staffie
- **Budget Détaillé** : Coûts mensuels et annuels
- **Carte des Parcs** : Espaces verts à proximité de Paris 2ème

## 🚀 Installation Locale

1. **Cloner le repository**
```bash
git clone https://github.com/ThibautMelen/baggy-staffie-guide.git
```

2. **Naviguer dans le dossier**
```bash
cd baggy-staffie-guide
```

3. **Ouvrir dans le navigateur**
```bash
# Sur Mac
open index.html

# Sur Windows
start index.html

# Sur Linux
xdg-open index.html
```
